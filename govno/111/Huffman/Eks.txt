1.НАЗНАЧЕНИЕ ПРОГРАММЫ 
Программа предназначена для сжатия данных путем кодирования текста по алгоритму Хаффмана.
Программа обеспечивает выполнение следующих основных функций:
ввод исходных данных и расчет характеристик;
реализация таблицы построения кода;
реализация дерева кодирования Хаффмана;
вывод промежуточного решения по итерациям и окончательного решения;
оценка стоимости реализации алгоритма по временным и объемным параметрам;
хранение исходных данных с возможностью их загрузки для повторной обработки;
хранение результатов решения с возможностью их повторной визуализации;
2.УСЛОВИЯ ПРИМЕНЕНИЯ
1.1.Программное обеспечение
Данная программа выполняется под управлением операционной системы Windows XP и последующих ОС этого семейства. Для запуска программы необходим файл Huffman.exe 
1.2.	Требования к аппаратуре
Для функционирования программы требуется ЭВМ со следующими характеристиками:
1.Процессор с тактовой частотой не менее 233 МГц.
2.Наличие не менее 64 Мб оперативной памяти.
3.Наличие не менее 2 MB свободного места на жестком диске для файлов приложения.
3.ВЫПОЛНЕНИЕ ПРОГРАММЫ
3.1Запуск программы
Для запуска программы необходимо запустить Huffman.exe
3.2Работа с программой
Программа состоит из главной формы на которой расположены пять вкладок. При запуске программы активной является вкладка «Ввод исходных данных» (рис. 1). Переключение между вкладками производится щелчком мыши по заголовку соответствующей вкладки или с помощью комбинаций клавиш Ctrl+Tab(переключение на следующую вкладку) и Shift+Ctrl+Tab(переключение на предыдущую вкладку).
 Рис. 1. Окно программы после запуска. Вкладка «Ввод исходных данных»
Описание работы с вкладкой «Ввод исходных данных»
Исходные данные загружаются в программу из внешнего файла указанного пользователем или с клавиатуры. Для загрузки исходных данных из файла необходимо нажать кнопку «Открыть» и в появившемся диалоговом окне указать путь к файлу с исходными данными. Для сохранение исходных данных в текстовый файл необходимо нажать кнопку «Сохранить как…» и в появившемся диалоговом окне указать имя сохраняемого файла, при необходимости подтвердить замену уже существующего файла. Для построения дерева кодирования Хаффмана и таблицы построения кодов, а также кодирования исходных по алгоритму Хаффмана используется кнопка «Расчет», при нажатии этой кнопки также происходит автоматический переход на вкладку «Результаты» (Рис. 5).
Описание работы с вкладкой «Промежуточные решения по итерациям»
На вкладке «Промежуточные решения по итерациям» отображаются шаги построения кодового дерева Хаффмана в виде таблицы. На каждом шаге два символа, имеющие наименьшие вероятности появления (соответственно наименьшие частоты появления), объединяются в один «псевдосимвол» с вероятностью (частотой) появления равной сумме вероятностей (частот) входящих в него символов. Объединяемые символы в программе выделяются красным цветом, а новый «псевдосимвол» - крупным шрифтом.
Рис. 2. Окно программы. Вкладка «Промежуточные решения по итерациям»
Описание работы с вкладкой «Кодовое дерево»
Рис. 3. Окно программы. Вкладка «Кодовое дерево»
Вкладка «Кодовое дерево» (Рис. 3) используется для визуального отображение построенного программой дерева кодирования Хаффмана. Перемещение “камеры” по изображению возможно с помошью набора кнопок расположенного в левой нижней части вкладки. Нажатие центральной клавиши возвращает “камеру” в начало кординат, а нажатие кнопки «U», «D», «L» или «R» приводит соответственно к смещению “камеры” вверх, вниз, влево или вправо.
Описание работы с вкладкой «Результаты»
На вкладке «Результаты» (Рис. 4) производится отображение результатов работы алгоритма: 
таблица кодирования (левая часть вкладки) содержащая все символы исходного сообщения, частоты их повторения и построенные для них коды Хаффман.
время затраченное программой на построение таблицы кодирования и кодового дерева, и на кодирование исходного сообщения.
количество бит занимаемых на диске исходным текстом при кодировании 1 Байт/символ и при кодировании по алгоритму Хаффмана.
Также на данной вкладке предоставляется возможность сохранение результатов решения и возможность повторной визуализации ранее сохраненных результатов. Для сохранения результатов работы алгоритма в файл необходимо нажать кнопку «Сохранение результатов работы алгоритма» и в появившемся диалоговом окне указать имя сохраняемого файла, при необходимости подтвердить замену уже существующего файла. 
Рис. 4. Окно программы. Вкладка «Результаты»
Для повторной визуализации ранее сохраненных результатов работы алгоритма из файла необходимо нажать кнопку «Загрузка сохраненных ранее результаты работы» и в появившемся диалоговом окне указать путь к файлу с сохраненными результатами работы алгоритма.
Описание работы с вкладкой «Закодированная строка»
На вкладке «Закодированная строка» (Рис. 5) отображает набор бит получаемый при кодировании исходного сообщения по алгоритму Хаффмана
